# Perfectionism Landing Page
A minimalist, interactive web page exploring the theme of perfectionism in work and study. Built with semantic HTML and custom CSS, this project features a fullscreen video background with an elegant hover-triggered message.
## Features
- Fullscreen background video with fallback poster image
- Semantic HTML5 structure with Open Graph meta tags
- Custom fonts (Raleway) and SVG favicon
- Pure CSS hover interaction – displays a witty message on title hover
- Accessible – muted, autoplay, and loop attributes for video; semantic headings
- Mobile-friendly viewport meta tag
